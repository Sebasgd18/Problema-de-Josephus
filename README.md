# Problema-de-Josephus
Problema de Josephus — Java NetBeans

Implementación del famoso Problema del Historiador Josephus utilizando una Lista Circular Simplemente Enlazada en Java.

El proyecto simula la eliminación secuencial de soldados en un círculo hasta encontrar el sobreviviente final, aplicando estructuras dinámicas de datos y algoritmos clásicos de sistemas computacionales.

Descripción del Problema

El problema de Josephus es un problema matemático e informático clásico donde:

Un grupo de personas se organiza en círculo.
Se elimina cada k-ésima persona.
El proceso continúa hasta que solo queda un sobreviviente.

En este proyecto:

Número de soldados (N) = 50
Salto de eliminación (k) = 5
